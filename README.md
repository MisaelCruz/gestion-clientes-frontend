# Sistema de Gestión de Clientes - Frontend Client 

Este repositorio contiene la interfaz de usuario moderna, dinámica y responsiva del Sistema de Gestión de Clientes. Es una Single Page Application (SPA) desarrollada con React y Vite que consume de forma asíncrona los servicios expuestos por nuestra API en el backend.

---

## Tecnologías Utilizadas

* **React** (Vite)
* **React Router Dom** (Manejo dinámico de rutas de navegación)
* **Axios** (Cliente HTTP para el consumo de la API REST)
* **Bootstrap 5** (Diseño responsivo y estilizado de componentes)

---

## Características Principales

* **Interfaz de Usuario Dinámica:** Renderizado eficiente de componentes en tiempo real utilizando hooks de React.
* **Navegación Fluida:** Control de vistas (Listado, Creación, Actualización) mediante rutas, garantizando una excelente experiencia de usuario (UX).
* **Consumo de API:** Integración con Axios para mapear dinámicamente las respuestas del backend en tablas y formularios interactivos.
